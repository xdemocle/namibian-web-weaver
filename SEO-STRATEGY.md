# LizWise SEO Strategy

## Current Implementation

The website currently implements the following SEO features:

- **React Helmet Async**: For managing document head metadata
- **Page-specific SEO components**: Each page has tailored metadata
- **Structured Data (Schema.org)**: Rich results for organization, products, and pages
- **Sitemap.xml**: For search engine crawling efficiency
- **Robots.txt**: With proper directives and sitemap reference
